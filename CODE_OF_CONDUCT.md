# 行動規範 / Code of Conduct

この場は **友朋共生（Yuuhou Kyousei — 友であり companion として共に生きる）** と **Ubuntu（私は、私たちがあるがゆえに在る）** を土台とします。

This community is grounded in *Yuuhou Kyousei* (living together as friends and companions) and *Ubuntu* (I am because we are).

## 私たちが約束すること / Our pledge

- 立場・国籍・専門・経験の違いにかかわらず、誰もが安心して持ち寄れる場を保つ。
- 意見には全力で向き合い、人格は決して攻撃しない。
- 反論を歓迎し、論破ではなく理解を目指す。

## 望ましい振る舞い / Encouraged

- 敬意ある言葉づかい。出典への謝意。
- 相手の真意を善意に解釈すること。
- 自分の誤りを認める率直さ。

## 許されない振る舞い / Unacceptable

- 侮辱、差別、ハラスメント、個人攻撃。
- 他者の貢献の無断・無出典での専有。
- 場を萎縮させる威圧的な言動。

## 運用 / Enforcement

問題があれば、リポジトリ管理者へ Issue（または管理者連絡先）で報告してください。管理者は、警告・編集・削除・参加停止を含む適切な対応を取ります。
